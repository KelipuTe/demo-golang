## leetcode

### 目录结构

lc开头的是leetcode题库的题目。
lcof开头的是剑指offer的题目。

### 分类标签

分类标签放在标题下面一行，用中括号括起来，不需要用逗号隔开。

```
[数组]

[二叉树]
[前序遍历][中序遍历][后序遍历][层序遍历]

[栈][队列]

[双指针]

[递归][二分查找][动态规划]
```

### go编码说明

- 指针用p开头
- 数组用arr开头，一维arr1，二维arr2
- 切片用sli开头，一维sli1，二维sli2
- map用map开头
- 内部方法用f0开头
- 栈用stack开头
- 队列用queue开头
